# My Portfolio

Welcome to my personal portfolio built with **ReactJS**, **TailwindCSS**, **GSAP**, and **EmailJS**!  
This project showcases my skills, projects, and experience with a modern, interactive, and responsive design.

## Features

- **Interactive Animations:** Smooth animations and transitions using GSAP.  
- **Responsive Design:** Fully responsive layout with TailwindCSS for mobile, tablet, and desktop.  
- **React Components:** Modular and reusable React components.  
- **Portfolio Showcase:** Display projects with images, descriptions, and links.  
- **Contact Form:** Send messages directly via EmailJS without backend setup.  
- **Smooth Navigation:** Easy navigation with smooth scrolling and intuitive UI.

## Technologies Used

- **ReactJS** – For building UI components.  
- **TailwindCSS** – For responsive and utility-first styling.  
- **GSAP (GreenSock Animation Platform)** – For animations and transitions.  
- **EmailJS** – For sending emails from the contact form.
